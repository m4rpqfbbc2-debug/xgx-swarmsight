Espresso foundations with ember signal. Every surface has warmth — no pure blacks, no clinical greys. Colours are named for materials, not abstractions.

Background hierarchy: Void → Espresso → Walnut → Clay. Never skip levels. Text on dark: Cream for headings, Bone for body, Patina for secondary, Stone for tertiary. Ember: Used only for interactive elements, active states, and data highlights — never decoratively. Signal Red: Reserved exclusively for destructive actions and error states.

Texture rules: Textures are layered via CSS background-image on top of background-color — they never replace colour. Opacity kept extremely low (0.01–0.04). Interactive surfaces use coarser textures (knurl, ribbed). Static surfaces use finer textures (leather, paper, canvas).
